# 中秋节非物质文化遗产专题网站 — 项目进度文档

## 项目概述

**项目名称：** 月满中华 · 情系中秋  
**项目类型：** 中秋节非物质文化遗产专题展示网站  
**技术栈：** 原生 HTML5 + CSS3 + Vanilla JavaScript（无框架依赖）  
**后端：** Node.js + Express（静态文件服务）  
**页面总数：** 10 个 HTML 页面  
**CSS 文件：** 4 个  
**JS 文件：** 8 个  

---

## 一、页面结构

### 1. 首页 `index.html`
全站门户，单页长滚动设计，共 12 个板块：
- **Section 1：Hero 全屏入口** — Canvas 粒子 + 月相视差层 + 标题动画
- **Section 2：中秋倒计时** — 动态倒计时（目标 2026年9月25日）+ 月相指示器
- **Section 3：千年溯源** — 节日起源概览（六节点迷你时间轴 + 视觉装饰）
- **Section 4：月宫传奇** — 四大神话传说卡片（嫦娥奔月/吴刚伐桂/玉兔捣药/月饼起义）
- **Section 5：民俗万象** — 八大民间习俗图标网格
- **Section 6：团圆味道** — 四大月饼流派轮播（广式/苏式/京式/滇式）+ 地方美食标签
- **Section 7：月下华章** — 六首中秋经典诗词错落展示
- **Section 8：光影中秋** — 中秋图集预览（6 格）
- **Section 9：非物质文化遗产** — 非遗传承历程（2006/2008/全球）
- **Section 10：月照四方** — 四国中秋文化卡片（日/韩/越/新加坡）
- **Section 11：千里共婵娟** — 祝福寄语互动（输入+放飞+祝福墙预览）
- **Section 12：Footer** — 全站导航 + 版权信息

### 2. 节日起源 `origin.html`
- Hero 区域（🧵 穿针引线主题）
- **直线时间轴** — 10 个时代卡片（上古→汉代→魏晋南北朝→唐代→宋代→元代→明代→清代→近现代→当代）
- 每卡片含：时代插图、徽章标签、年份、标题、两段正文（~200字）、1-2 条古籍引用
- SVG 直线轴 + 滚动驱动绘制动画
- IntersectionObserver 驱动卡片激活 + 背景切换
- `data-journey="origin"` 月华流转主题

### 3. 民间习俗 `customs.html`
- Hero 区域
- **Flip Card 网格** — 12 个翻转卡片（悬停翻转查看详细描述）

- **四大分类详情板块**（每板块含多个 Tab 切换卡片）：
  - 类别一：敬天礼月（赏月/祭月/观潮）
  - 类别二：灯火祈福（燃灯/烧塔/舞火龙/玩花灯）
  - 类别三：饮食品味（吃月饼/赏桂饮酒/博饼）
  - 类别四：游艺民俗（兔儿爷/走月亮）
- 每张详情卡片含 5 个 Tab：历史渊源/民俗流程/地域差异/文化寓意/当代传承

### 4. 神话传说 `legends.html`
- Hero 区域 + 月宫背景装饰
- **文化体系标签筛选：** 全部 / 中华汉族神话 / 中华民间传说 / 东亚月亮传说
- **Flip Card 网格** — 9 个翻转卡片：
  - 中华汉族神话：嫦娥奔月 / 吴刚伐桂 / 玉兔捣药
  - 中华民间传说：月饼起义 / 月下老人 / 天狗食月
  - 东亚月亮传说：辉夜姬（日）/ 韩国月兔 / 阿贵与榕树（越）
- **三大详情板块**（每张卡片含 6 个 Tab）：

- **模块4：食俗雅趣** — 6 项折叠面板（拜月供品/桂花佳酿/中秋啜螺/分饼之仪/赏月宴饮/团圆饭仪）

## 五、技术亮点

1. **月华流转系统** — `moon-journey.js` 为每个页面/阶段提供独特的背景色彩方案，通过 CSS 自定义属性实现平滑过渡
2. **SVG 时间轴绘制** — `origin.js` 动态计算卡片位置生成直线路径，滚动驱动 stroke-dashoffset 实现"绘制"动画效果
3. **暖色调独立主题** — `food.html` 和 `fame-detail.html` 通过 CSS 变量完全覆盖默认深蓝配色，呈现暖色美食主题
4. **3D 翻转卡片** — 美食页和传说页使用 `transform: rotateY(180deg)` + `backface-visibility` 实现悬停翻转
5. **Tab 切换系统** — 习俗页和传说页的详情卡片内嵌 5-6 个 Tab 面板，纯 CSS + JS 实现
6. **IntersectionObserver 滚动驱动** — 全站使用 IO 替代传统 scroll 事件实现滚动揭示、背景切换、卡片激活
7. **CSS 自动滚动** — 美食页时间轴使用 `@keyframes` 实现纯 CSS 无缝循环滚动
8. **粒子系统** — 双套粒子（背景漂浮粒子 + Canvas 粒子连线），营造星空氛围
9. **孔明灯放飞** — 动态创建 DOM 元素 + CSS 动画实现祝福放飞效果
10. **响应式设计** — 所有页面均适配桌面/平板/手机三端
